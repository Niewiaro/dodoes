  • Create structured to-dos with ease
  • Use UUIDs to uniquely track and manage each item

Reviews

What developers are saying about DoDoes API...

FastAPI is surprisingly pleasant to work with

As someone who's worked with backend frameworks, I was genuinely impressed by FastAPI. It's fast, elegant, and extremely developer-friendly. The automatic docs, request validation, and Pydantic integration make building APIs feel almost effortless. DoDoes uses FastAPI just right - it's modern, clean, and really satisfying to work with.

Review by Niewiaro
Still holding love for Django REST Framework

Even though DoDoes API impressed me with FastAPI's performance and async support, I'll admit it - my heart still beats for Django REST Framework. The ORM, class-based views, and ecosystem are unmatched. That said, DoDoes proves that if you need a lightweight and blazing-fast API, FastAPI can be the perfect tool for the job.

Review by Jakub
A simple and clean API, just how it should be

What I like most about DoDoes API is how it gets out of your way. The endpoints are predictable, the docs are interactive, and everything works as you'd expect. Whether you're building a todo app or integrating it with your own frontend, this API keeps things efficient. It could use more advanced filtering, but for most use cases, it's more than enough.

Review by Kuba

What is DoDoes API?

  • DoDoes API is a modern, developer-first interface for managing tasks and to-dos.

  • You can create, update, organize, and complete your tasks using clean RESTful endpoints, with full support for user accounts, priorities, deadlines, and more.

  • Built with FastAPI, it offers automatic documentation, token-based authentication, and a clear data structure designed for productivity apps, automation tools, or personal dashboards.
